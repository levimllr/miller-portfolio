Giffer Reborn

A virtual Arduino coding platform.

Contributions:

  • 400+ use to learn hardware programming
  • Simulates light/motor output and sensor input
  • .gif renders from Javascript-intepreted C++
  • Generates exercises in 1 click with source code
  • Can be simply hosted through GitHub

Tools:

  • JS + Ace, JSCPP, Clipboard.js, and highlight.js
  • CSS3 + Bootstrap
  • Arduino Uno, Mega
  • GitHub Pages
  • HTML5

Digital Music Box

A mechatronic song composer.

Contributions:

  • Allows for different notes, instruments, tempos.
  • Capacitavely senses movable metal pins.
  • Mixes tracks with a MIDI sound module.
  • Manages data with Arduino + Raspberry Pi.
  • WIP: Touch-screen interface.

Tools:

  • Python + Kivy
  • Arduino Mega & Raspbery Pi + SlushEngine
  • 200+ fabricated, COTS, and stock parts
  • ODrive motor, capactivate sensors, RGB LEDs
  • MIDI protocol
